# References

This repository does not host paper PDFs directly. For Pattern Database background, refer to the original publications or official bibliographic pages.

- Richard E. Korf and Ariel Felner. *Disjoint Pattern Database Heuristics*. Artificial Intelligence, 2002.
- Ariel Felner, Richard E. Korf, and Sarit Hanan. *Additive Pattern Database Heuristics*. Journal of Artificial Intelligence Research, 2004.

The implementation notes in this repository summarize how disjoint additive PDBs are used in the 15-Puzzle solver; they do not redistribute the full papers.
